Under Air India's GDS misuse policy, "Churning" refers to the process of making repeated bookings for the same sector or overlapping sectors for the same passenger. This can occur through a single Passenger Name Record (PNR) or across multiple PNRs within the same Global Distribution System (GDS) or even across various GDSs by an agent.

For bookings made in Economy Class, up to four churns are permitted per passenger per segment. After that, an Agent Debit Memo (ADM) of US$10 will be charged for each additional churn. In the case of Premium class bookings, the threshold is set at ten churns. The parameters used to determine churn include the same origin, destination, passenger name, and IATA code.
