Under Air India's GDS misuse policy, "Churning" refers to the process of making repeated bookings for the same sector or overlapping sectors for the same passenger through a Passenger Name Record (PNR) or across multiple PNRs, either on the same Global Distribution System (GDS) or across various GDSs by an agent. 

For bookings made in Economy Class, a maximum of four churning instances are permitted. After that, an Agent Debit Memo (ADM) of US$10 will be charged for each additional churn per passenger per segment. For Premium class bookings, the threshold is set at ten instances of churning. The parameters used to determine churn include the same origin, destination, passenger name, and IATA code.
